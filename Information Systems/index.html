Teoria da Informação

Processamento de dados em informação

Transformação de conjuntos de dados em um sistema semanticamente compreensível.

Tipologia dos sistemas de informação

Sistema de informação gerencial (SIG)

  • Processo de transformação de dados em informações
  • Coleta, valida, executa operações, transforma, armazena e apresenta informações

Sistema de apoio à decisão (SAD)

  • Podem ser classificados como as hoc quando desenvolvidos para uma situação única e institucional para situações periódicas.
  • Principal característica: uso de modelos e dados de diferentes fontes

Componentes do SAD

  • Interface: Banco de conhecimento; linguagem de ação; linguagem de apresentação.
  • Dados: Devem possuir idade adequada, viés, serem confiáveis e relevantes ao processo decisório.
  • Modelo: Simplificação do fenômeno a fim de entender seu comportamento.

Data Warehouse

Grande banco de dados contendo dados históricos resumidos em diversos níveis de detalhamento.

Data mining

Exploração e análise, por meio automático ou semi-automático de grande quantidade de dados para descobrir modelos e regras simplificadas .
